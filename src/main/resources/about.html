Codewars plugin

This is not an official codewars.com plugin. This is an open source project created by a fellow warrior. The source code can be found at https://github.com/ansis-m/CodeWarsPlugin.

I am still working on the project and the plugin is quite buggy. Please report bugs and send suggestions to maleckisansis@gmail.com. Also, consider contributing to the project on github. The codebase is very raw and there is a lot of room for improvement.

The plugin currently supports katas in:

  • Java in Intellij community and ultimate
  • Kotlin in Intellij community and ultimate
  • Groovy in Intellij community and ultimate
  • Python in PyCharm, Intellij community and ultimate
  • JavaScript and Typescript in Intellij ultimate and WebStorm
  • Ruby in Intellij ultimate and RubyMine

To start a kata in any of the supported languages you first need to create a project (or at least a module) for that specific language. Next, sign-in to codewars.com in the embedded browser. Chose a kata, select language and press "train". Navigation to the training page triggers creation of the kata files which will be opened in your Intellij workspace. Each kata will have a separate directory that will contain also a test file (for local testing) and some metafiles for the normal functioning of the plugin. Edit the work/test file and once you are done submit the local file for testing via "test" or "attempt" buttons. When you pass the "attempt" you will be able to submit the solution.

You can start several/many katas in the same project/module. If the current project contains several katas, you can jump between them by using the drop-down menu on the bottom. The completed katas will have a green checkmark. You can also delete all the kata files.

Happy coding!
